# 情報源調査メモ

2026-04-22 時点で、落語スケジュール集約アプリの初期ソースとして有力な公開情報源を整理したメモです。

## 採用方針

- まずは公式サイト優先
- HTML 上で開催日、会場、出演者が取れるページを優先
- 個人ブログ、転載サイト、SNS 単独は初期段階では除外

## 有力ソース

### 1. 落語協会

- URL: https://www.rakugo-kyokai.jp/
- 取れそうな情報:
  - 本日の寄席
  - 定席番組
  - 落語会情報
- 使い道:
  - 落語協会系の定席と独演会の両方を補完できる
- 備考:
  - トップページ上で `本日の寄席 2026年04月21日` を確認
  - `落語会情報` への導線がある

### 2. 落語芸術協会

## 収集の分け方

### A. 定席系

- 落語協会
- 落語芸術協会
- 浅草演芸ホール
- 新宿末廣亭
- 池袋演芸場
- 国立演芸場

特徴:

- 継続性が高い
- HTML の規則性が比較的ある
- 毎月の席単位で更新される

### B. 独演会・ホール落語系

- 落語協会の落語会情報
- 落語芸術協会の落語会情報
- 各会場のイベントページ

特徴:

- 件数は多いが、表記ゆれが大きい
- 会場や主催者によって項目が違う
- 個別ページをたどる必要がある

## 個人利用向けの現実的なスタート

最初は以下だけで十分です。

1. 落語協会
2. 落語芸術協会
3. 四大定席の公式サイト
4. 国立演芸場

これで「寄席」と「主要な公式落語会」の基盤ができます。

## 実装上の注意

- 日付表記が `2026年4月20日`、`4/21`、`4月21日〜25日` などで混在する
- `昼の部 / 夜の部` を 1 イベントにまとめるか分けるかを先に決める
- 出演者変更が多いので、詳細は `最終確認日時` を見せる
- スクレイピングはキャッシュを挟み、1 サイトへのアクセス回数を抑える
- 著作権表記があるサイトでは本文全文を持たず、要約済みの構造化データだけ保存する
